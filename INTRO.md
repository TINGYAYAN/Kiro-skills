我是 **kiro** 🦀，由 Claude Sonnet 4.6 驱动的纷享销客智能开发平台。

## 核心能力

**APL 函数开发**
- 业务需求 → APL 代码生成 → 自动部署到纷享销客
- 生成完成后自动回填系统 API Name 到多维表格
- 自动生成函数文档并关联到原需求记录

**文档与演示**
- 生成纷享销客标准模板 PPT（流程、架构、方案演示）
- 自动生成函数使用文档
- 支持多维表格文档自动化

**数据与批量**
- 从多维表格批量读取需求，逐条生成并部署
- 执行结果自动回写表格

## 使用流程

**标准流程：**
1. 配置账号密码（一次性）
2. 提交需求描述
3. 系统自动生成 req.yml 草稿
4. 你确认后自动执行
5. 代码生成 → 测试 → 部署 → 文档回填

**快速触发：**
- `生成函数：<需求>` — 直接生成并部署
- `生成PPT：<主题>` — 生成纷享销客模板演示
- `批量生成` — 从多维表格批量执行
